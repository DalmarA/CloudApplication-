What Is Cloud Applications

A cloud app is an application that operates in the cloud. Cloud apps are considered to be a blend of standard Web applications and conventional desktop applications.

Servers Vs Clients


Servers are powerful computers dedicated to managing file servers , print Servers, or serving up resource for clients. Servers are usually headless meaning there are no peripheral devices, this is done to lower electricity consumption because servers are usually on 24/7. Clients are PC, laptops and mobile devices which users run applications and rely on servers for resources, such as files, devices, media, and even processing power.

Self-Hosting Cloud Application

Own Cloud

Is a Cloud App that allows people to host their own cloud solution with their own equipment and internet connect. This is useful for people who don't wat to pay a monthly fee.

Sonarr

This is a cloud application used to automatically tracks, downloads and sorts tv shows with the use of Usenet Groups and/or Torrents after the series is added from the Web UI or compatible Browser Extensions.

Counch Potato

Is a cloud application that automatically tracks, downloads and sorts Movies with the use of Usenet Groups and/or Torrents when they have been tagged with the IMDB browser extension or from the Web UI.

Headphones

Is a cloud application that automatically tracks, downloads and sorts music of artists added to the database from either Usenet or Torrent.
