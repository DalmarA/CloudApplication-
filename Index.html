What Is Cloud Applications

A cloud app is an application that operates in the cloud. Cloud apps are considered to be a blend of standard Web applications and conventional desktop applications.


Relation to Business and technology


Servers Vs Clients


Servers are powerful computers dedicated to managing file servers , print Servers, or serving up resource for clients. Servers are usually headless meaning there are no peripheral devices, this is done to lower electricity consumption because servers are usually on 24/7. Clients are PC, laptops and mobile devices which users run applications and rely on servers for resources, such as files, devices, media, and even processing power.

Self-Hosting Cloud Application


Own Cloud

Is a Cloud App that allows people to host their own cloud storage solution with their own internet connect.

Sonarr

This is a cloud application used to automatically tracks, downloads and sorts tv shows with the use of Usenet Groups and/or Torrents after the series is added from the Web UI or compatible Browser Extensions.

Counch Potato

Is a cloud application that automatically tracks, downloads and sorts Movies from Usenet Groups and/or Torrents that have been added with the IMDB extension or the Web UI.

Headphones

Is a cloud application that automatically tracks, downloads and sorts music from Usenet groups and/or torrent after artists is added to web UI.

Media Servers

Own Cloud, Sonarr, Couch Potato and Headphones succesfully uses the cloud to retrive content but inorder to view the content with server based processing a media server application is needed.


Emby

This is a very feature packed free open source media server which also allows casting to UPNP/DLNA devices and chrome cast, roku, etc.


Plex

This is application is older and has more users but in order to view content on your server you would need an Plex app or view as network file system on the client.

IOT Devices for playback of content

If your goal is to view content on your tv with out an PC then Emby would work best.
